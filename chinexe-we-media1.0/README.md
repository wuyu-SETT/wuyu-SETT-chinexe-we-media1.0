<h1><br><h1 align="center"><br>2.“自媒体生产流水线”
</h1>
首先我们要解决的是做一个自媒体有哪些流程
<br><br>1️⃣思考与准备（对游戏自媒体和自己有一个基本的认识，设备，软件，账号等等准备）
<br><br>2️⃣生产内容（素材，剪辑，标题）
<br><br>3️⃣发布和运维（平台选择，粉丝互动）
<br><br>4️⃣复盘和调整（平台自带数据工具或第三方工具）
<br><br>5️⃣重复执行和找一个好导师（一个好的导师比老师更重要）
</h1>

<h1 align="center"><br>3.设备和录制软件</h1>
N卡用户英伟达app推荐自带的n卡录制（有即时回放很好用）A卡的话有ReLive方便快捷，OBS全能免费，强大，可录屏，可直播

<h1 align="center"><br>4.平台选择</h1>
<br>简单介绍一下适合做游戏自媒体的平台
<br>不同平台的用户群体，内容形式，推荐机制，变现方式等等都不太一样，我的推荐是简单了解一下，一到三个平台一起做，哪个平台有突破，就投入更多精力优化，多学习
<br><br>1️⃣抖音，流量最多竞争最激烈，全年龄段，短视频，快节奏居多
<br><br>2️⃣b站，社区氛围好，年轻人多，适合几乎所有类型，特别是做深度内容，粉丝粘性高
<br><br>3️⃣快手，短视频，全年龄段，快节奏居多
<br><br>4️⃣视频号，全年龄段，推荐机制偏社交
<br><br>5️⃣YouTube，油管，全球化，适合几乎所有类型，竞争大

</h1>

<h1 align="center"><br>5.账号</h1>
自媒体平台都会给用户打1-5个左右的标签，以便精准推送流量，我们可以通过刷我们想做的同类内容让机制给我们打一个标签，比如刷三角洲，机制就给你打上三角洲的标签，除了这个之外还有年龄，消费区间等等，就随便刷刷，别疯狂收藏点赞，平台可能会判定你是干黑灰产的，然后限流你这个账号
<br><br>
账号包装
<br>
名字，好记、好搜，名字涉及关键词，比如SETT（三角洲行动教学），SETT（三角洲行动）等等，那我取最强最大最好（三角洲行动）或多个关键词的名字流量不就有了吗？想多了，最强最大最好这一类属于违规词
<br><br>
头像，突出风格，搜同类型博主参考一下
<br><br>
简介，10秒让用户了解你，搜同类型博主参考一下，注意敏感词
<br><br>
头图，突出风格，搜同类型博主参考一下<br><br>
对标账号:明确诉求,找到相似的达人,参考成功模式。抖音搜索热点宝<br>
做抖音的必须了解热点宝，后面很多的地方会用上，这没什么好讲的，自己去搜，看一下就明白了
</h1>

<h1 align="center"><br>6.面相用户定位（做什么、给谁看）</h1>

决定三件事：

1️⃣做什么类型（游戏教学、陪玩氛围、剧情讲解，直播卖货…）

2️⃣给什么人看（新手玩家、休闲玩家、某游戏粉丝，lsp…）

3️⃣视频风格，标签（搞笑、治愈、情绪、专业，抽象…）

人设与 IP 包装

定位越清晰，受众越精准，转化率越高，平台算法推流越精准

</h1>

<br>带相关的标签，比如某某游戏新出的武器，热梗等等，符合用户的兴趣<br>
实用选题方法<br><br>
找同行热门视频<br><br>

看同类型博主的评论区<br><br>

<h1 align="center"><br>8.思路和剧本（我认为不是必要的）（对于前期来说保持兴趣和找到方向很重要）</h1>

我把它分为两部分，先做宏观的内容思路，抽象脚本，然后再做具体的视频剧本<br><br>
宏观思路就是你想传达的核心信息，有趣的游戏时刻，你是谁，你想传达什么<br><br>
具体的脚本就是找一个载体将它具体的描述出来，然后想想怎么把它做的有趣，专业等等让用户看下去，根据平台特性优化<br><br>
去热点宝借鉴爆款视频类型或刷刷教学视频，能给你带来一些思路<br>
脚本一般分三点<br><br>

吸引人的开头，封面，文案<br><br>
输出内容<br><br>
总结<br><br>
那要怎么吸引人呢，网上都有详细的教程，这里不过多赘述，因为我觉得说太多太繁琐了，你看又看不懂，这东西没什么好学的，你自己做几个视频发布复盘看看数据就会做了，在错误中学习，这里主要是告诉你该做什么和一些容易踩坑的点<br>
</h1>

素材的形式有很多，主要就是自己录制的视频素材，以及一些第三方的音频素材，GIF表情包，ae特效等等<br><br>
如果你愿意无偿为他人分享请联系我！

<h1 align="center"><br>10.剪辑（自媒体成败关键！）</h1>

剪辑不难但很吃时间，创意<br>
剪辑说白了通过通过音乐或视觉冲击等方式控制好节奏感和观众情绪，先吸引观众看你的视频，然后再想办法留住观众，然后输出内容给他<br>
一你的定位是什么，吸引什么样的观众看你的视频，根据这些制定你视频的标题和封面，前三秒很重要，一般有三种方式<br><br>
1️⃣开头高潮，最简单有效，通常是视觉冲击和信息冲击结合<br><br>
2️⃣抛出一个足够吸引人的问题，比如为什么99%的营销号喜欢用<br><br>为什么99%的****开头
如果洋务运动成功了中国会怎么怎么样<br><br>
3️⃣制造反差，比如uzi跳枪<br><br>
留住观众，通常就两种形式，一通过内容吸引观众，二通过控制节奏感和情绪留住观众
<br><br><br>
“剪辑流水线”<br><br>
1️⃣看素材 ，选素材<br><br>
2️⃣想一下视频的大致结构，粗剪，把大概的模子做出来<br><br>
3️⃣对齐音乐节奏<br><br>
4️⃣加转场和音效<br><br>
5️⃣加字幕顺便复盘<br><br>
6️⃣细节优化<br><br>
7️⃣上传后看反馈<br><br>
8️⃣不断改标题，封面，开头重新发布<br><br>
</h1>
